MAPTIVE - HOSTING AND ARCHITECTURE STEPS
======================================

1. PROJECT ARCHITECTURE
-----------------------
- FRONTEND: React.js (Vite + TypeScript)
- BACKEND: FastAPI (Python)
- DATA/AUTH: Firebase (Firestore & Auth)
- AI ENGINE: Grok (xAI) or Gemini (Google)

STRUCTURE:
User Browser <-> Frontend (Vercel) <-> Backend (Render/Railway) <-> AI APIs
                          ^
                          |
                   Firebase (Real-time)

2. STEP-BY-STEP HOSTING
-----------------------

STEP 1: FIREBASE SETUP
- Create project at console.firebase.google.com
- Enable Google Auth.
- Create Firestore Database.
- Copy keys to your environment.

STEP 2: BACKEND HOSTING (FastAPI)
- Use a platform like Render.com or Railway.app
- Connect your GitHub repo.
- Root directory: /backend
- Start command: uvicorn main:app --host 0.0.0.0 --port $PORT
- Add environment variables if needed.

STEP 3: FRONTEND HOSTING (React)
- Use Vercel.com or Netlify.com
- Connect your GitHub repo.
- Root directory: /frontend
- Build command: npm run build
- Output folder: dist
- Add all VITE_ environment variables.

STEP 4: CONNECTION
- Ensure the Frontend points to the Backend URL.
- Ensure the Backend CORS allows the Frontend URL.
