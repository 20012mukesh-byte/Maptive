MAPTIVE: ENTERPRISE NETWORK DIGITAL TWIN SOLUTION
================================================

1. PROJECT OVERVIEW
------------------
Maptive is a high-fidelity, real-time network infrastructure visualization and management platform. 
It bridges the gap between physical architecture and logical network monitoring by creating a 
"Digital Twin" of campus networks. 

Unlike traditional static dashboards, Maptive uses AI to convert architectural PDFs and natural 
language prompts into interactive, data-driven network maps.

2. THE PROBLEM
--------------
Modern university and enterprise campuses suffer from "Network Blindness":
- IT Admins have physical blueprints but lack real-time connection to logical failures.
- Monitoring software is often purely text-based or uses generic, uninspired diagrams.
- Translating physical site changes to network layouts is manually intensive.

3. THE SOLUTION (MAPTIVE)
-------------------------
Maptive provides an "Arctic Glass" visual interface where the network is alive:
- **Architecture Synchronization**: Upload a building floor plan PDF, and Maptive uses AI (Grok Vision) to automatically identify and place networking hardware (Routers, Switches, PCs) on a canvas.
- **Natural Language Orchestration**: Use a chat interface to "build" or "modify" the network (e.g., "Add three PCs to the library switch").
- **Live Failure Physics**: When a real-world device reports a failure via Firebase telemetry, its visual counterpart pulses red, and connection edges "snap" like severed cables.
- **AI-Powered Diagnostics**: An AI Analyst (Grok/Gemini) reads chronological telemetry logs to explain the *root cause* of outages rather than just reporting them.

4. KEY FEATURES
---------------
- **AI Architect Engine**: Generates deterministic Cisco-style topologies from text or PDF scripts.
- **Real-Time Telemetry Pipeline**: Integrated with Firebase Firestore for sub-second visual updates.
- **Interactive Cisco CLI**: A futuristic command-line overlay that highlights network components in real-time as you type diagnostic commands.
- **Self-Healing Visualization**: Visualizes redundant paths and automatic rerouting using dashed blue "packet" animations.
- **Arctic Glass UI**: A premium, minimalist design using Framer Motion physics and Backdrop-Blur glassmorphism.

5. TECHNICAL STACK
------------------
- **Frontend**: React 19, Vite, TypeScript, Tailwind CSS v4, Framer Motion, React Flow.
- **Backend**: FastAPI (Python), NetworkX (Graph Modeling), Uvicorn.
- **Real-Time Layer**: Firebase Auth & Cloud Firestore.
- **AI Model Integration**: xAI (Grok-1), Google Gemini 2.0.
- **Data Export**: jsPDF (Topology Reports), Custom Cisco Config Text.

6. USE CASES
------------
- **University IT Departments**: Mapping dorms and labs to real-time status monitors.
- **Datacenter Management**: Visualizing rack failures and bandwidth bottlenecks.
- **Network Training**: Simulating failures in a safe, visual environment for students and junior engineers.
